# Tesler Documentation Project
[Live Demo](http://idocs.tesler.io/)

### Prerequisites:
#####
* java 8
* maven
* node.js
* npm
* git
* docker
* docker-compose

## Getting started with docker
* install dependencies and build front-end
```
mvn clean install -PUI
```
* start environment
```
docker-compose up -d
```
* run application
```
press green button in Application.java
```
